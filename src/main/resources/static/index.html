新手流程（推荐）：
1) 先准备部件图（透明 PNG，按身体部位拆分）
2) 导入后进入“建骨”，把骨头从根到末端画出来（例如：身体→上臂→前臂→手）
3) 在“绑定部件”把每张 PNG 挂到对应骨骼上，再拖拽摆正
4) 在“做动作”给骨头旋转打关键帧，点击播放预览
5) 导出工程 JSON（后续可用于 Godot 运行时）
